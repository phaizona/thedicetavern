# 🎲 The Dice Tavern

A fantasy-themed dice rolling website built with vanilla HTML, CSS, and JavaScript.

**The Dice Tavern** is a lightweight browser-based dice roller designed for tabletop RPG players, Dungeon Masters, and fantasy enthusiasts. Instead of a plain utility, the project aims to recreate the feeling of rolling dice inside a cozy medieval tavern lit by candlelight.

## 🎮 Live Demo

Play it here:

https://phaizona.github.io/thedicetavern/

## ✨ Features

* Standard RPG dice:

  * d4
  * d6
  * d8
  * d10
  * d12
  * d20
  * d100
* Custom dice creation (up to 1000 faces)
* Advantage rolls
* Disadvantage rolls
* Drop Lowest rolls
* Roll modifiers
* Multiple dice rolls at once
* Critical Success and Critical Failure detection
* Roll history tracking
* Animated dice rolling effects
* Medieval tavern-inspired interface

## 🏰 Design Goals

The project was created as a fun fantasy-themed utility rather than a realistic dice simulator. Every visual element—from the wooden tavern background and candlelight effects to the custom dice icons—was designed to make rolling dice feel more immersive and enjoyable.

## 🛠 Built With

* HTML5
* CSS3
* Vanilla JavaScript

No frameworks, libraries, or external dependencies were used for the core functionality.
